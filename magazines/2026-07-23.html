Инцидент с безопасностью: OpenAI и Hugging Face наткнулись на проблемы при оценке моделей

OpenAI and Hugging Face address security incident during model evaluation

OpenAI и Hugging Face сообщили об инциденте безопасности, произошедшем во время оценки их языковых моделей. Иронично, но попытка Hugging Face использовать ИИ для анализа логов не сработала, так как требовала подачи реальных команд атаки. Этот случай подчеркивает сложности обеспечения безопасности в системах ИИ, особенно для продуктов, работающих с конфиденциальными данными.

Историческое решение: Anthropic заплатит $1,5 млрд за пиратские книги, использованные для обучения Claude

Judge approves $1.5B Anthropic settlement for pirated books used to train Claude

Суд утвердил мировое соглашение, по которому Anthropic (разработчик модели Claude) выплатит $1,5 миллиарда за использование пиратских книг при обучении своего ИИ. Это знаковое событие формирует прецедент в вопросах авторских прав и ИИ, вызывая дебаты о том, достаточно ли одноразовой выплаты или же нужна постоянная система роялти за 'переваривание' чужих идей.

«Режим кода» для LLM: сокращение затрат на 99,2%

Code mode yields a 99.2% cost reduction in our systems

Компания Agent Swarm поделилась опытом, как использование 'режима кода' (программно-управляемого вызова инструментов) для LLM позволило сократить операционные расходы на 99,2% по сравнению с обычными промптами на естественном языке. Это демонстрирует мощный способ оптимизации и удешевления работы ИИ-систем, делая их более эффективными за счет использования детерминированных языков для оркестровки задач.
