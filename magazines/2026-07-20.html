Claude Code переехал на Bun, написанный на Rust

Claude Code uses Bun written in Rust now

Anthropic, создатель Claude, перевел свою модель Claude Code на использование JavaScript-рантайма Bun, который сам написан на Rust. Это решение, вероятно, направлено на повышение производительности и стабильности, поскольку Bun известен своей скоростью и эффективностью. В обсуждениях критикуют предыдущее решение использовать Zig и выражают смешанные чувства по поводу постоянных изменений в стеке.

OpenAI reduces Codex Model Context Size from 372k to 272k

OpenAI неожиданно сократил размер контекстного окна своей модели Codex с 372 тысяч до 272 тысяч токенов. Это изменение может повлиять на производительность и возможности модели для задач, требующих обработки больших объемов информации, например, при анализе кода или длинных документов. В комментариях высказывается беспокойство по поводу потери детализации при уменьшении контекста, несмотря на потенциальную оптимизацию.

LoRA Speedrun: рейтинг скорости тонкой настройки LLM

LoRA Speedrun – a public wall-clock leaderboard for fine-tuning techniques

Запущен проект LoRA Speedrun — публичная таблица лидеров, оценивающая скорость и эффективность различных методов тонкой настройки (fine-tuning) LLM с использованием LoRA (Low-Rank Adaptation). Это позволяет сравнивать подходы к обучению моделей не только по качеству, но и по времени, что критически важно для исследователей и разработчиков. Обсуждение в комментариях затрагивает ценность оптимизации ресурсов и важность "идти на меньшее" в противовес тренду на создание гигантских моделей.
